## 핵심 결정

- **Draw 동작**: 키노트 → 필드 전체 (8장) 묘지로 + 8장 새로 draw (v1 ResetCards 패턴)
- **드로우 풀 (_drawPile)**: 게임 시작 시 _deck → shuffle. Draw로 소진되면 ShuffleAll() (묘지 합쳐 재셔플).
- **더미 덱**: 데이터셋 미도착이므로 SeedDummyDeck()로 24장 random suit/rank 생성 (effect 없는 빈 카드). 시각 표시 + Poker hand 판정 용. 데이터셋 도착 시 JSON loader로 교체.
- **CardSystem과의 관계**: CardSystem._field에 CardData만 채움. _activeEffects는 Registry-driven (별도). 효과 발동은 등록된 dummy 3종만, field 8장은 시각 표시.
- **초기 draw**: Start()에서 KeyNoteProcessedEvent 구독 + 8장 draw + SetField + FieldChangedEvent publish.
- **EventBus 사용**: KeyNoteProcessedEvent 구독, FieldChangedEvent publish.
