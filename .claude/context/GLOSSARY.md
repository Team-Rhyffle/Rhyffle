# Rhyffle — 용어집

> rhyffle 도메인 용어 정본. 새 용어는 여기에 추가.

## 게임 구조

- **라인 (line)**: 노트가 떨어지는 세로 슬롯. 명시적 "lane" 명칭은 코나미 특허 회피 위해 사용 불가 — "카드와 1:1 대응되는 슬롯" 개념으로 우회. v2 2차에서 **8개**, v2 1차에서는 7개였음
- **슬롯 (slot)**: 라인과 동의어. 각 라인에 카드 1장씩 대응
- **칸**: 1개 라인이 3칸으로 세분화. 판정에 영향 없고 채보 다양화용 시각 구분만
- **필드**: 카드들이 드로우되어 점수 효과를 발동하는 영역. v2 2차 8슬롯
- **덱**: 게임 시작 시 카드 더미. v2 2차 = **40장**. (v2 1차는 56장 = 52 포커 + 4 조커)
- **묘지 (graveyard)**: 필드에서 소모된 카드 이동 영역. v2 2차 = 단일 더미 (FIFO 폐기). v2 1차는 FIFO
- **드로우**: 덱에서 빈 필드 공간에 카드를 놓음. v2 2차 = 게임 시작 시 8장, 키 노트 처리 시 8장 교체
- **리바이브**: 덱이 비면 묘지 전체를 셔플 후 덱으로 이동
- **셔플 키**: 카드 구역 왼쪽 버튼. 콤보 게이지 일정 이상 시 활성화, 클릭으로 즉시 새 드로우

## 노트

- **탭 노트 (BasicNote)**: 도착 순간 터치
- **홀드 노트 (HoldNote)**: 시작-끝 사이 터치 유지. 긴 홀드는 사이 판정 노트들이 BPM 따라 존재. JSON에서 같은 `count` 끼리 묶임
- **슬라이드 노트 (SlideNote)**: 노트 통과 시 화면 터치만 유지
- **플릭 노트 (FlickNote)**: 탭 처리 + 스와이프. 상/하/좌/우 4방향. 좌우 플릭은 5/12 결정으로 도입 (고점 재미 우선)
- **키 노트 (Key Note)**: 노트 종류와 무관, 카드 교체 트리거. 곡당 최소 8개~최대 15개

## 판정

- **Perfect / Great / Good / Miss**: 4단계. 각각 노트 점수의 120% / 100% / 70% / 0%
- 배속 (r): 0.1~3.0 지원. 판정 범위는 노트 세로 길이 h와 r로 계산 — Perfect = (1 + 0.2r)·h 등
- **콤보**: 연속 hit 카운트. 셔플 키 활성화 조건 / 점수 보너스에 사용 (정확한 룰 미정)

## 점수 모드

- **Plain Mode**: 카드 효과 없이 진행. 1,000,000 / 총 노트 수 = 노트별 점수. 합산 = 노트 점수 × 판정 배율. 만점 = 1,200,000
- **Challenge Mode**: 카드 효과 포함
  - **클래스 점수**: 각 카드 class에 따른 가산 (예: First +10, Premier +50)
  - **시너지 점수_1**: 멤버/시즌/테마/클래스 카테고리 조건 만족 시 합연산 추가
  - **시너지 점수_2**: 일정 매수 이상 시 곱연산 추가
  - 총점 = { **Σ** (노트 + 클래스 + 시너지_1) × 시너지_2 } × 판정 배율

## 카드 (COSMO Objekt 기반, v2 2차)

- **포토카드 (Photocard)**: MODHAUS 소속 아티스트 IP 기반 수집형 카드
- **Objekt**: COSMO 앱의 NFT 포토카드. 게임 내 카드의 원본 정의
- **아티스트 (artist)**: 그룹 이름. `tripleS`, `ARTMS`, `idntt` 등
- **멤버 (member)**: 영문명. `SeoYeon`, `DoHun`, `Heejin` 등
- **시즌 (season)**: 발행 시점 정보
  - tripleS/ARTMS: `Atom01 → Binary01 → Cream01 → Divine01 → Ever01 → Atom02 ...` 순환
  - idntt: `Spring25 → Summer25 → Autumn25 → Winter25 → Spring26 ...` 순환
- **컬렉션 번호 (collectionNo)**: `숫자3자리 + A/Z` (예: `120Z`)
  - 백의 자리 = 희귀도 (대략적)
  - `A` = 오프라인 실물 + 디지털 NFT 쌍 / `Z` = 디지털 NFT 단독
- **클래스 (class)**: 카드 등급 (희귀도와는 다름)
  - tripleS/ARTMS: `First < Double < Special < Premier < Motion`. 디코 5/21 결정에 따라 **Premier → Special 변경** 진행 중 (페이지 미반영)
  - idntt: `Basic < Event < Special < Unit < Motion`
  - 데모 제외: `Welcome`, `Zero`, `Motion` 등 예외 class
- **오브젝트 번호 (objektNumber)**: 같은 종류 카드 중 몇 번째로 발행된 사본인지 (예: `#00123`)
- **카드 도감 ID (key)**: `<artist> <season> <member> <collectionNo>` 조합 (예: `tripleS Atom01 ChaeYeon 101Z`)

## 카드 매커닉 (v2 1차 / 26736a63 페이지) — ❌ 데모 미포함 (5/24 한울 확정)

> 이 섹션 전체 (포커 매커닉 / 카드군 / 양면 / 능력 / 반물질) 데모에 포함되지 않음. 향후 프로덕션 단계 부활 가능성으로 정의는 보존.

- **문양 (suit)**: 스페이드 / 다이아몬드 / 하트 / 클럽 (v2 1차 포커 매커닉)
- **랭크 (rank)**: A, 2~10, J, Q, K (+조커)
- **카드 등급 (rarity)**: Common < Rare < Epic < Legendary (Card DB 데이터소스는 + Basic 5단계, UI/UX 페이지는 4단계 — 정본 미확정)
- **양면 카드**: 플릭 노트 영향 받음. 한 번 뒷면 되면 앞면 복귀 X. 뒷면 = 마이너스 점수 + 강력 특수능력
- **반물질 (antimatter)**: 유저 재화. 양면 카드 뒷면 활성화에 소모
- **카드군 (card group)**: 카드의 분류 단위. 일반 카드 = 카드군 X, 특수 카드는 카드군 보유. 카드군 12종 컨셉 (장미전쟁/공격/지원/수비/버서커/카오스)
- **고유능력 (unique ability)**: 카드군/카드 종류 고정 능력. 변경 불가. 카드군 시너지 목적
- **일반능력 (general ability)**: 카드 획득 시 랜덤 부여. 일반능력 변경 가능 (재화 소모). 희귀도 높을수록 가짓수 多

## 강화 (v2 2차 신규)

- **화음 (harmony)**: 같은 아티스트, 같은 등급, **다른 멤버** 카드로 강화. 10단계
- **공명 (resonance)**: 같은 아티스트, 같은 등급, **같은 멤버** 카드로 강화. 10단계
- 화음/공명 둘 다 **실패 확률 0** (성공 보장)
