## 족보 밸런싱 고려사항

### 1. 해당 족보가 얼마나 자주 나타나는지

- 상위 족보와 관계없이 나타나는 횟수 고려
- 족보를 달성하는 난이도를 확인할 때 용이 (아무래도 최상위 족보가 될 수 있는 기회다 보니)

### 2. 해당 족보가 얼마나 최상위 족보를 유지하는지

- 족보가 많이 나타나도 상위 족보의 유무에 따라 해당 족보의 점수가 반영 안될 수 있으니 고려

## 족보 밸런싱_250703 ver

1. 족보 우선 순위:
    
    원 페어 < 투 페어 < 트리플 < 스트레이트 < 플러시 < 쓰리 페어 < 풀 하우스 < 포 카드
    
    < 레인보우 < 스트레이트 플러시 < 더블 트리플 < 럭키 세븐
    

소수점 아래 둘째자리에서 반올림

1. 원 페어: x1.011
2. 투 페어: x1.02211
3. 트리플: x1.0999 = x1.1
4. 스트레이트: x1.12345 = x.1.1
5. 플러시: x1.11
6. 쓰리 페어 (순위 버프 예정): x1.554433
7. 풀 하우스: x1.32
8. 포 카드: x4.444
9. 레인보우: x13.7654321
10. 스트레이트 플러시: x15.0
11. 더블 트리플: x33.2211
12. 럭키 세븐: x100.0
13. 현재 족보는 5장 핸드 기준으로 세워진 족보로 (쓰리 페어 제외) 신규 족보의 추가 필요성
    - 아무래도 5장 핸드일 때와 7장 핸드일 때의 족보 달성 난이도 차이가 클 것으로 폐기해야 할 족보들이 있을 수 있음. (개인적으론 원 페어)
    - 5장 핸드일 때는 만들 수 없지만, 7장 핸드일 때 만들 수 있는 족보도 있음.
        
        (ex. 포 카드 + 트리플 조합 or 스트레이트 7장 버전으로)
