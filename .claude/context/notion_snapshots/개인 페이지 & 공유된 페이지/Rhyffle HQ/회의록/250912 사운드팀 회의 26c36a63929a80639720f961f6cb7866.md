## 주제 : 게임 효과음 / 배경음악 정하기

### 배경음악

- 로딩 화면 ~ 타이틀 화면 (10초) - 우주 배경 맞춰서 잔잔한 곡
- 메인 화면 (1분) - 웅장한 느낌이면 좋을듯 합니다…!
- 플레이 결과 화면 (10초) - 짧게 비트감있는 곡 (세이)
- 덱 구성 / 카드 관리 화면 (1분) - 체류시간이 긴 화면이라 반복은 많되 자극적이지 않은 곡 (세이)
- 상점 - 세이님 곡 ☆ Press [E]nter ☆ 1분까지로 사용

로딩 / 메인 → 시온님

플레이 결과 / 덱 카드 관리 → 세이님
