### 기획

내구도 삭제

카드 56장 → 52 + 4 (문양별로 조커)

게임 시작시 덱을 랜덤으로 셔플하고, 7장씩 10초마다 새로 드로우. 덱 모두 소진시 묘지에서 다시 셔플 후 덱으로 보내 다시 드로우

카드 효과 종류 (드로우 시 / 온필드 / 파괴 시 )

족보 종류 (문양 / 숫자 / 문양 + 숫자)

따라서 카드군은 문양 / 숫자 / 문양 + 숫자 세 타입을 지정하는 쪽으로 확정

우선 하트+다이아 (레드 카드), 짝수 (2, 4, 6, 8, 10) 카드들을 지정하는 카드군 만들어보기로
