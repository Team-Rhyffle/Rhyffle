## WHY

### 지나치게 복잡한 덱 빌딩 난이도

1. 랭크/문양 + 특수효과까지는 괜찮음.
2. 여기에 내구도가 더해지게 되면, 언제 카드가 파괴되고 다음 카드를 드로우해서 점수를 극대화할지 계산이 지나치게 어려워짐.
3. 내구도의 높음/낮음 밸런싱 잡기 너무 힘듦…

### 필드의 7장 카드마다 다른 드로우 타이밍의 복잡함

1. 내구도가 다르므로 서로 다른 타이밍에 파괴되고, 다음 카드가 드로우되면서 시각적으로도, 점수 시스템적으로도 너무 복잡함.
2. 효과가 추가될수록 서로 충돌가능성 높음.

## 기존안 유지 의견

1. 머리를 쓰는 맛 → 훨씬 복잡하므로 우리 타겟 유저층 (리듬게임 오타쿠) 공략하기 좋음
2. 내구도
    1. 범위 1~30 → 
    2. 카드 A 고유능력을 가진
    3. 내구도 / 문양 / 랭크 → 랜덤해야 ? 하나로 고정?
    4. 내구도 조정 8로 +- 2정도씩 조정 / 문양은 고정 / 랭크는 설정
    5. 고유한 이름 카드는 장수제한 필요
3. 효과가 다양해서 단조롭지 않다 → 카드군 종류 / 기획이 많아짐 → 재미의 고점이 높다
