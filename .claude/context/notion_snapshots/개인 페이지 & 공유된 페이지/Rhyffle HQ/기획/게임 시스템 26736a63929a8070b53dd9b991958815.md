## 2. 노트

1. 탭 노트 (영상의 흰색 노트)
    1. 가장 기본적인 노트로, 노트가 도착하는 순간 화면을 터치해서 처리
2. 홀드 노트 (파란색 선 노트)
    1. 시작 지점에서부터 터치를 유지하고, 끝나는 지점에서 터치를 뗌
    2. 기본적으로 시작과 끝 지점에 판정 1회씩이 존재하고, 보다 긴 홀드의 경우에는 곡의 BPM따라 사이 판정 (내부 노트)들이 추가로 존재함.
3. 슬라이드 노트 (파란색 노트)
    1. 노트가 지나가는 동안 누를 필요 없이 화면이 터치된 상태이면 처리
4. 플릭 노트 (초록/빨강 노트)
    1. 탭 노트 처리방식에 더해 누르는 순간 위/아래로 스와이프
    2. 플릭 노트 처리 시, ‘뒷면’이 있는 양면 카드는 앞뒷면이 전환된다

+) 키 노트 (기본 색상 + 반짝임 효과?)

- 노트의 종류와는 상관없이, 새 카드가 드로우되는 타이밍을 유저에게 보여주고 시스템적으로 정하기 위한 장치.
- 곡의 길이에 따라 최소 7개, 최대 15개의 키 노트가 존재. (최소 → 덱 1바퀴 / 최대 → 덱 2바퀴)
- 키 노트의 판정과는 상관 없이 기존 필드의 7장 소모 및 새로운 7장 드로우는 일어난다.

## 3. 판정

노트 당 4가지의 판정 범위가 존재함. 홀드 노트의 경우 각각의 내부 노트들에 별개로 판정 범위가 존재

![image.png](%EA%B2%8C%EC%9E%84%20%EC%8B%9C%EC%8A%A4%ED%85%9C/image.png)

노트 오브젝트의 세로 길이가 h, 배속 정도가 r일 때, (r = 1일 때 시스템 기본 배속. 0.1~3.0 까지 일단 지원)

Perfect (하늘색) 범위

- (1 + 0.2*r) * h
- 노트 점수의 120% 획득

Great (연두색)

- (1 + 0.4*r) * h
- 노트 점수의 100% 획득

Good

- (1 + 0.7*r) * h
- 노트 점수의 70% 획득

Miss: 노트를 놓침

- (1 + r) * h
- 노트 점수의 0% 획득

세부 수치는 실제 플레이해본 후 조정 필요합니다.

## 4. 점수 시스템

<aside>
💡

카드 효과를 받는 Challenge Mode와 카드 없이 진행되는 Plain Mode로 나뉩니다.

</aside>

### Plain Mode 기준 설명

- 1,000,000점을 기준으로, 노트의 총 개수로 나누어 노트별 점수가 책정됨.
    - 나누어진 값이 정수가 아닌 경우, 소수점 첫째자리에서 반올림 후 곡의 첫번째 노트에 남는 값 더하거나 뺌.
- ex) 총 노트가 1444개인 곡의 경우, 1000000 / 1444 = 692.52
    - 노트당 693점
    - 첫번째 노트는 1000000-693*1443 = 1점
        - 가장 극단적인 경우로, 다른 경우에서는 이정도로 편차 안남.
- 합산 점수는 노트 점수 * 판정 배율로 계산.
    - 이론상 만점 = 1,200,000점

### Challenge Mode 설명

- Plain Mode와 기본 노트 점수 계산까지는 모두 동일
- 랭크 점수 : 각 카드의 앞면 점수 (ex. J면 +11점, A이면 +1점, 조커이면 0점) 가 노트 점수에 더해진다.
- 새로운 7장이 등장할 때 (게임 시작 or 키 노트 처리) 마다 족보를 계산한다.
- 족보 점수 : 곡 플레이 종료 시, 완성한 족보의 상위 n개 종류에 따라 최종 점수에 곱연산으로 보너스 점수가 주어진다.
- 즉, 총 합산 점수는 { **Σ** (노트 점수 + 카드 점수) * 판정 배율 } * (족보 점수) 가 된다.
- 자세한 내용은 본 문서 6. 족보 시스템 에서 설명.

- 모든 카드 효과 및 점수는 왼쪽의 카드 내용을 처리한 후 오른쪽 카드로 넘어간다
- 한 노트가 여러 라인에 걸쳐있는 경우에도 동일

### 구성 요소

- 덱 : 게임 시작 시 유저가 소유하는 카드 더미로, 56장으로 구성됨
    - 52장은 포커의 기본 덱 구성 (4개 문양 x 13개 랭크) 을 따른다
    - 4장은 조커 스페이스로, 각 문양 (스페이드, 다이아, 하트, 클럽) 별 조커 1장씩을 편성한다.
    - 드로우가 진행되며 덱이 빈 경우, 묘지의 모든 카드를 셔플한 후 덱으로 이동시킨다 (리바이브)
- 드로우 : 덱에서 빈 필드의 공간에 카드를 놓는다
    - 게임 시작 시, 기본적으로 7개의 드로우 실행
    - 카드는 ‘키 노트’ 가 등장할 때 마다 다음 7장의 set으로 교체된다.
        - 현재 필드의 7장을 모두 묘지로 이동 / 덱에서 다음 7장을 드로우
- ~~묘지 드로우 : 특수 효과 혹은 양면카드를 통해 덱이 아닌 묘지에서 드로우 할 수 있다~~
- 필드 : 카드들이 드로우 된 후, 리듬 게임 점수에 효과를 발동하는 영역
    - 총 7개의 슬롯이 존재함
    - 덱, 묘지의 카드는 보너스 점수 / 족보에 영향을 미치지 못하고, 오직 필드의 카드들만으로 계산이 이루어진다.
- 묘지 : 필드에서 소모된 카드가 이동하는 영역
    - 묘지의 구조는 FIFO (가장 먼저 파괴된 카드가 묘지의 맨 위에 존재, 이후 파괴될 시 묘지 아래부터 쌓임) 를 따른다.

## 6. 족보 시스템

- 게임 시작 혹은 키 노트 처리 이후 새로운 7장이 등장할 때 마다 (7장씩의 단위를 ‘**페이즈**’ 라고 한다), 족보를 계산한다.
- 족보 목록은  [2. 족보](%EC%B9%B4%EB%93%9C%20%EC%8B%9C%EC%8A%A4%ED%85%9C%2026736a63929a80fc95d7c507104e6772.md) 참고
- 한 번의 페이즈에서 가장 높은 족보만 ‘족보 목록’에 기록된다.
- 곡 플레이 완료 이후, 족보 목록에 기록된 족보들 중 가장 높은 족보 5개에 따라 최종 보너스 점수가 부여된다.
    - 단, 같은 종류의 족보는 1개씩만 최종 보너스 점수에 기여할 수 있고, 중복 시 다음으로 높은 족보가 가산된다.
    - 완성된 족보가 4개가 되지 않을 경우에도 현재 완성된 족보들만으로 보너스 점수를 계산한다.
