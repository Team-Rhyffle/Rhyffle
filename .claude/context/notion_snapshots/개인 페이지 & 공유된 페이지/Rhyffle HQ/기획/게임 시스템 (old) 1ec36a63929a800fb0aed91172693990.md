## 2. 노트

</aside>

1. 탭 노트 (흰색 노트)
    1. 가장 기본적인 노트로, 노트가 도착하는 순간 화면을 터치해서 처리
2. 홀드 노트 (선 노트)
    1. 시작 지점에서부터 터치를 유지하고, 끝나는 지점에서 터치를 뗌
    2. 기본적으로 시작과 끝 지점에 판정 1회씩이 존재하고, 보다 긴 홀드의 경우에는 곡의 BPM따라 사이 판정 (내부 노트)들이 추가로 존재함.
3. 슬라이드 노트 (파란색 노트)
    1. 노트가 지나가는 동안 누를 필요 없이 화면이 터치된 상태이면 처리
4. 플릭 노트 (초록/빨강 노트)
    1. 탭 노트 처리방식에 더해 누르는 순간 위/아래로 스와이프
    2. 플릭 노트 처리 시, ‘뒷면’이 있는 양면 카드는 앞뒷면이 전환된다

## 3. 판정

노트 당 4가지의 판정 범위가 존재함. 홀드 노트의 경우 각각의 내부 노트들에 별개로 판정 범위가 존재

![image.png](%EA%B2%8C%EC%9E%84%20%EC%8B%9C%EC%8A%A4%ED%85%9C/image.png)

노트 오브젝트의 세로 길이가 h, 배속 정도가 r일 때, (r = 1일 때 시스템 기본 배속. 0.1~3.0 까지 일단 지원)

Perfect (하늘색) 범위

- (1 + 0.2*r) * h
- 노트 점수의 120% 획득

Great (연두색)

- (1 + 0.4*r) * h
- 노트 점수의 100% 획득

Good

- (1 + 0.7*r) * h
- 노트 점수의 70% 획득

Miss: 노트를 놓침

- (1 + r) * h
- 노트 점수의 0% 획득

세부 수치는 실제 플레이해본 후 조정 필요합니다.

### 구성 요소

- 덱 : 게임 시작 시 유저가 소유하는 카드 더미로, 60장으로 구성됨
    - 52장은 포커의 기본 덱 구성 (4개 문양 x 13개 랭크) 을 따른다
    - 8장은 조커 스페이스로, 유저 본인이 원하는 카드를 자유롭게 구성할 수 있다. (단, 중복 불가)
    - 드로우가 진행되며 덱이 빈 경우, 묘지의 모든 카드를 셔플한 후 덱으로 이동시킨다 (리바이브)
- 드로우 : 덱에서 빈 필드의 공간에 카드를 놓는다
    - 게임 시작 시, 기본적으로 7개의 드로우 실행
    - 카드의 내구도가 다 소모되어 묘지로 이동해 빈 슬롯이 발생한 경우, 해당 슬롯에 대해 드로우를 실행한다.
- 묘지 드로우 : 특수 효과 혹은 양면카드를 통해 덱이 아닌 묘지에서 드로우 할 수 있다
    - 묘지 드로우를 통해 필드로 나온 카드는, 원래의 내구도를 모두 회복한다.
- 필드 : 카드들이 드로우 된 후, 리듬 게임 점수에 효과를 발동하는 영역
    - 총 7개의 슬롯이 존재함
    - 덱, 묘지의 카드는 보너스 점수 / 족보에 영향을 미치지 못하고, 오직 필드의 카드들만으로 계산이 이루어진다.
- 묘지 : 내구도를 다 소모하거나, 특정 조건을 통해 파괴된 카드가 이동하는 영역
    - 묘지의 구조는 FIFO (가장 먼저 파괴된 카드가 묘지의 맨 위에 존재, 이후 파괴될 시 묘지 아래부터 쌓임) 를 따른다.

## 5. 점수 시스템

<aside>
💡

카드 효과를 받는 Challenge Mode와 카드 없이 진행되는 Plain Mode로 나뉩니다.

</aside>

### Plain Mode 기준 설명

- 1,000,000점을 기준으로, 노트의 총 개수로 나누어 노트별 점수가 책정됨.
    - 나누어진 값이 정수가 아닌 경우, 소수점 첫째자리에서 반올림 후 곡의 첫번째 노트에 남는 값 더하거나 뺌.
- ex) 총 노트가 1444개인 곡의 경우, 1000000 / 1444 = 692.52
    - 노트당 693점
    - 첫번째 노트는 1000000-693*1443 = 1점
        - 가장 극단적인 경우로, 다른 경우에서는 이정도로 편차 안남.
- 합산 점수는 노트 점수 * 판정 배율로 계산.
    - 이론상 만점 = 1,200,000점

### Challenge Mode 설명

- Plain Mode와 기본 노트 점수 계산까지는 모두 동일
- 각 카드의 앞면 점수 (ex. J면 +11점) 가 노트 점수에 더해진다.
- 현재 필드의 카드가 하나라도 바뀔 시, 그때마다 족보를 계산한다.
    - 곡 플레이 종료 시, 완성한 족보의 상위 n개 종류에 따라 최종 점수에 곱연산으로 보너스 점수가 주어진다.
- 즉, 총 합산 점수는 { **Σ** (노트 점수 + 카드 점수) * 판정 배율 } * (족보 점수) 가 된다.
