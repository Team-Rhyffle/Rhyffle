# 메인 (홈) 화면

소유자: 김한울
생성 일시: 2025년 7월 27일 오후 10:54

### Background

개발팀에서 현재까지 개발한 화면들을 하나로 합치는 작업을 진행하고 싶다고 하셔서, 다른 디자인 요소들 (카드 디자인, 에셋 등등) 보다 메인화면 디자인을 먼저 진행해주셨으면 합니다!

메인 화면 디자인 완성은 당장 급하지 않기에, 각 기능들의 진입점 위주로만 구성하고, 추후 다른 디자인 작업이 끝나고 개발 마무리 단계때 한번 크게 손봐야 할 듯 합니다.

## 메인화면 레퍼런스 / 구성요소

![image.png](%EB%A9%94%EC%9D%B8%20(%ED%99%88)%20%ED%99%94%EB%A9%B4/image.png)

일단은 현재 필요한 진입점 버튼들이 적기에, 위와 같은 레퍼런스를 가져왔습니다.

추후 가챠나 이벤트 배너, 다양한 모드들이 추가될 경우 전면적으로 수정해야 하므로, 디자인 크게 공들이지 말아주세요…! 순수히 개발 및 초기버전 발행용으로 쓰기 위한 UI 배치입니다.
