## 컨셉

Reference: https://ko.wikipedia.org/wiki/%EC%BF%B5%ED%91%B8_%ED%8C%AC%EB%8D%94 / https://ko.wikipedia.org/wiki/%EC%82%AC%EC%9B%90%EC%86%8C%EC%84%A4

- 스승(멘토)를 만나 수련을 통해 능력을 기르고, 마스터를 만나 능력을 극대화시키는 컨셉인 대형 카드군
- ACE를 주인공으로 하여 ACE가 필드에 나오는 순간부터 효과 적용
- ACE가 특정 카드(3, 5, 7, 9, 조커)를 만나게 되면(홀수 카드가 필드에 나타나게 되면) 스택 생성
    - 홀수 카드로는 최대 4개까지 스택을 쌓을 수 있음. (일반적인 경우)
    - 조커를 만나게 되면 스택 여부와 상관없이 스택이 5가 됨. (특수한 경우로 스택 5는 혜택 대폭 증가)
- 주인공에게는 시련이 있어야 하는 법. ACE가 문양 상관없이 짝수 카드(2, 4, 6, 8, 10)를 만나게 되면 스택 모두 소멸
- 시련을 극복한 주인공에게는 더 큰 보상이 기다림. → 빌런을 만난 횟수에 비례하여 최종 효과 증폭
- 주인공이 특정 마스터(K, Q, J)를 만나는 순간 스택에 비례하여 원소에 맞는 혜택 적용
- 하트 - 불 / 클로버 - 흙 / 다이아몬드 - 공기 / 스페이드 - 물 로 원소 설정
    - 하트의 경우 족보 배율 상승 혜택 (%로 증가)
    - 클로버의 경우 콤보 비례 추가 점수 혜택
    - 다이아몬드의 경우 노트 당 추가 점수 혜택
    - 스페이드의 경우 현재 점수에 추가 점수 혜택 (%로 증가)

빌런 : 어떠한 원소의 주인공 효과가 활성화된 시점에서 빌런을 만나게 되면 스승 스택 초기화 및 주인공 효과 비활성화

- 처치한 빌런의 개수에 따라 마스터를 만났을 때의 최종 효과 배율이 달라짐. (적용할까 말까)

- 각 문양의 에이스 별로 같은 문양의 K, Q, J 카드를 만나야 함. ex) 스페이드 에이스는 스페이드 K, Q, J를 만나야 혜택 적용
