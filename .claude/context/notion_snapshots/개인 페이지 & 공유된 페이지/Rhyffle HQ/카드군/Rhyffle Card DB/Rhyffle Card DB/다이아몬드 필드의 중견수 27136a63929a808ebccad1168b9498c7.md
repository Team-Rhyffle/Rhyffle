# 다이아몬드 필드의 중견수

고유능력: 외야수 - 이 카드는 “외야수” 특성을 갖는다.
외야의 지휘자 - 필드의 “외야수” 특성을 갖는 카드의 수에 따라 아래의 효과를 얻는다.
1장 : 실책 - 이 카드가 처리하는 노트에 대한 판정은 항상 ‘Miss’ 로 취급한다.
2장 : 다이빙 캐치 - “외야수” 카드들이 처리하는 노트에서 ‘Miss’ 판정이 나올 경우, 이를 ‘Good’ 판정으로 취급한다.
3장 : 낙인 “철벽 외야” 를 획득한다.

문양: 다이아몬드
