# 다이아몬드 필드의 유격수

Note: +73: 08베이징 올림픽 끝내기 병살플레이를 만든 박진만(7), 고영민(3) 등번호 레퍼런스
고유능력: 내야수 - 이 카드는 “내야수” 특성을 갖는다.
내야의 지휘자 - 필드의 “내야수” 특성을 갖는 카드의 수에 따라 아래의 효과를 얻는다.
1장 : 실책 - 이 카드가 처리하는 노트에 대한 판정은 항상 ‘Miss’ 로 취급한다.
2장 : 하이점프캐치 - 필드의 “내야수” 카드들이 처리하는 노트에서 ‘Good’ 이상의 판정이 나올 경우, 이를 모두 ‘Perfect’ 판정으로 취급한다.
3장 : 더블플레이 - 필드의 모든 “내야수” 카드들의 랭크 점수에 보너스 +73점을 부여한다.
4장 : 낙인 “철벽 내야” 를 획득한다.
